# Agent Formateur – Guide d’utilisation

Cet agent « formateur » est conçu pour répondre à vos questions de façon pédagogique, en s’adressant à un apprenant sans background informatique. Il résume chaque échange dans un fichier .md dans le dossier Cours, en expliquant :

- L’objectif du cours
- Le comment et le pourquoi des actions
- Les points à retenir

## Fonctionnement
1. Posez une question à l’agent formateur.
2. L’agent répond de façon claire et simple.
3. Un fichier .md est créé dans le dossier Cours pour résumer l’échange.

## Exemple d’utilisation
- Vous : « Qu’est-ce qu’un dossier à la racine du projet ? »
- L’agent : Réponse pédagogique + création d’un fichier résumé dans Cours.

N’hésitez pas à poser toutes vos questions, même les plus simples !
